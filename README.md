## 👨‍💻 GojaeDoo | 프론트엔드 개발자
안녕하세요! 저는 **UI/UX 최적화**와 **반응형 웹 애플리케이션** 개발을 전문으로 하는 **프론트엔드 개발자** **GojaeDoo**입니다.  
**React, Next.js, TypeScript**를 기반으로 **효율적이고 직관적인 웹 애플리케이션**을 구현합니다.  
끊임없이 **성능 최적화**, **유연한 상태 관리**, **모던 웹 기술**을 학습하고 있습니다.
